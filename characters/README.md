# Characters Folder

This folder contains structured character profiles designed for use in AI roleplay, storytelling, worldbuilding, or simulation prompts.

These characters are formatted to be easily used in any large language model (LLM), including systems like ChatGPT, Gemini, Claude, and similar tools.

---

## Purpose

The goal of this folder is to provide:
- Consistent, reusable character sheets
- Easy integration into AI chats or writing workflows
- Clear personality, appearance, and narrative structure
- Ready-to-use profiles for storytelling or simulation

---

## How to Use These Characters

You can use these character profiles in any LLM by:

### 1. Copy & Paste Method
Simply copy a full character file and paste it into your prompt.

Example:
> “Use this character as a roleplay partner / story NPC / simulation subject.”

---

### 2. System / Role Prompting
You can assign a character as a role:

Example:
> “You are Kenji Takahashi. Respond and behave according to the following profile…”

---

### 3. Multi-Character Scenarios
You can combine multiple characters for:
- Dialogue scenes
- Story interactions
- Simulation environments
- Relationship dynamics

## Character Structure

Each character typically includes:

- Basic Information (name, age, origin)
- Physical Appearance
- Background & History
- Personality Traits
- Speech Style
- Current Situation
- Core Themes

This structure is designed to help models maintain consistency in behavior and tone.

---

## Guidelines for Best Results

To get the most out of these characters:

- Be clear about the scenario (roleplay, story, simulation, etc.)
- Specify tone (serious, emotional, realistic, cinematic, etc.)
- Avoid contradictory instructions within the same prompt
- If using multiple characters, define relationships clearly
