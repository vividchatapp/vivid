# Stories Directory 📖

This folder contains plain text files generated from your AI interactions. These files are designed to be used as drafts or final versions of stories created during your sessions.

## 🎬 How it Works

Vivid Chat includes specific features to facilitate immersive storytelling:

### 1. Story Mode (`.mode story`)
When you switch to story mode using the `.mode story` command, the bot changes how it handles conversation history:
- **Assistant-Only Context**: Instead of sending the full chat history (which includes your commands and meta-talk), the bot filters the context sent to the LLM. 
- **Narrative Focus**: It primarily sends previous **assistant responses** and your latest prompt. This ensures the AI stays focused on the narrative flow rather than getting distracted by previous technical commands.

### 2. Saving the Story (`.story save [name]`)
Once you have a conversation you are happy with, you can compile it into a single document:
- **Extraction**: Use the command `.story save MyStoryName`.
- **Content**: The bot extracts every response generated by the **assistant** (the LLM) in the current session.
- **Format**: These responses are joined together with double newlines and saved as a `.txt` file right here in this folder.

## 📝 Editing
The resulting `.txt` files are standard text documents. You can open them in any editor to refine the prose, adjust the pacing, or combine multiple chapters into a larger work.
